LicitTrack

Projeto desenvolvido no Mestrado Profissional em Tecnologia da Informação do IFPB, com foco em soluções analíticas aplicadas ao setor público. O LicitTrack surgiu para solucionar a dificuldade enfrentada por gestores públicos, pesquisadores e fornecedores ao acessarem informações sobre licitações. Os dados disponíveis no Portal Nacional de Contratações Públicas (PNCP) são fragmentados, inconsistentes e pouco acessíveis para análise. A proposta do projeto foi consolidar essas informações em uma plataforma analítica única e interativa.

A solução técnica envolveu a construção de um pipeline de dados em Python com extração via múltiplos endpoints da API do PNCP. Para acelerar o processo, implementei paralelismo com ThreadPoolExecutor. Após a extração, os dados JSON foram padronizados e armazenados em um banco MongoDB, estruturado para consultas analíticas e suporte a dados aninhados.

Com os dados modelados, desenvolvi um painel frontend em React + Tailwind CSS. O dashboard oferece visualização por indicadores estratégicos (valor licitado, evolução anual), táticos (modalidades e órgãos contratantes) e operacionais (status de licitações, itens mais licitados). O sistema é responsivo, interativo e pensado para autoatendimento (self-service BI).

Uma das seções do painel permite a análise dos fornecedores participantes nas licitações, com visualização do número de empresas por ano, tipo jurídico e ranking dos maiores participantes. Essa funcionalidade fornece insights valiosos para tomada de decisão e análise de mercado público.

O sistema também oferece um painel para a análise dos itens mais licitados, permitindo observar quais produtos ou serviços concentram maiores volumes em contratos públicos. Os dados são filtráveis por localidade e ano, com totalização e ordenação automática dos valores licitados.

O LicitTrack foi registrado no Instituto Nacional da Propriedade Industrial (INPI), o que reforça sua originalidade e caráter inovador. Esse reconhecimento formal fortalece a relevância do projeto como solução tecnológica aplicada à gestão pública e à transparência.

O projeto visa facilitar o acesso e análise de dados públicos, promovendo transparência e eficiência. O desenvolvimento do LicitTrack envolveu o uso de diversas tecnologias modernas.
