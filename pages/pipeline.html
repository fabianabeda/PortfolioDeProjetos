Integração de Dados Ambientais

Projeto acadêmico desenvolvido durante o Mestrado Profissional em Tecnologia da Informação (IFPB), com o objetivo de integrar dados ambientais para análise de impactos urbanos. Os dados utilizados provinham do AQICN (qualidade do ar) e do MapBiomas (uso do solo). Apesar de serem públicos, os dados estavam em formatos distintos e complexos de integrar.

Construímos um pipeline automatizado e escalável, orquestrado com Apache Airflow utilizando PythonOperator para estruturar o fluxo ETL. O ambiente foi totalmente containerizado com Docker, promovendo reprodutibilidade e escalabilidade para diferentes contextos.

Os dados foram armazenados em um Data Lake construído com MinIO, organizados em camadas Raw (bruta), Staging (intermediária) e Trusted (confiável). Essa estrutura possibilitou versionamento, rastreabilidade e governança adequada. O processamento foi realizado com Apache Spark, convertendo JSONs em arquivos Parquet otimizados. Em seguida, os dados foram carregados em um banco PostgreSQL com modelagem orientada à análise.

Para a visualização, conectamos o Power BI ao banco de dados e desenvolvemos dashboards com KPIs ambientais como níveis de poluição por município, áreas degradadas e comparações por ano. A interface foi pensada para usuários públicos, com navegação intuitiva e exploratória.

As integrações com APIs públicas foram realizadas diretamente no Airflow. O AQICN foi acessado via REST e o MapBiomas via GraphQL. As conexões foram autenticadas e documentadas como mapbiomas_login e minio no ambiente do pipeline.
