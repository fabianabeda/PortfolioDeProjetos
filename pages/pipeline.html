Pipeline de Dados Ambientais

Projeto do mestrado profissional PPGTI - IFPB que integra dados do MapBiomas e AQICN para análise ambiental. A proposta foi desenvolver um pipeline completo e automatizado com governança de dados, processando informações de desmatamento e qualidade do ar para posterior visualização em dashboards interativos.

A orquestração das tarefas foi feita com Apache Airflow, usando PythonOperator para organizar a DAG de extração, transformação e disponibilização dos dados. Todo o ambiente foi containerizado com Docker, garantindo escalabilidade e portabilidade do projeto.

Os dados foram armazenados em buckets do MinIO, simulando uma arquitetura de Data Lake com camadas: Raw (dados brutos), Staging (dados intermediários tratados) e Trusted (dados prontos para consumo). Essa organização permite governança e reuso dos dados. Após o processamento com Apache Spark, os dados foram convertidos para o formato Parquet e salvos no PostgreSQL. Assim, os dados puderam ser modelados para consumo analítico com ferramentas como Power BI.

A visualização final no Power BI permite entender os municípios com maior área de vegetação suprimida, assim como os estados mais impactados. Os gráficos e tabelas apresentam dados atualizados e filtráveis por ano, promovendo maior transparência ambiental.

O uso do MapBiomas API no pipeline permitiu o download direto dos dados por meio de integração GraphQL. A conexão foi configurada no Airflow como mapbiomas_login, enquanto o MinIO foi acessado pela conexão minio.
