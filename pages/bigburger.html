BigBurger

O BigBurger é uma aplicação web Full Stack em desenvolvimento que simula um sistema de gestão de pedidos para hamburguerias. O objetivo do projeto é aplicar na prática conhecimentos de desenvolvimento moderno com React e Node.js, como parte das atividades do curso de Tecnologia em Sistemas para Internet.

A aplicação terá um frontend feito em React, com rotas dinâmicas, gerenciamento de estado global com Context API e estilização com Styled Components. Já o backend está sendo construído com Node.js + Express, usando MongoDB como banco de dados e implementando autenticação segura com JWT e Bcrypt.

No futuro, o sistema permitirá a criação de usuários, gerenciamento de pedidos, atualização do cardápio e histórico de entregas. Também está nos planos a containerização com Docker para facilitar a implantação e testes locais.
